به دلیل این که حجم ویدیوی رکورد شده ممکن است خیلی زیاد شود، روشی را پیشنهاد می کنم تا پس از ضبط ویدیو/صوت، با اجرای یک اسکریپت پایتون، فایلها بدون دردسر و به صورت خودکار، کامپرس شوند.
برای تحویل غیر آنلاین (ارسال ویدیو/صوت) مراحل زیر را گام به گام دنبال کنید:
1. برای تسک هایی که انجام داده اید، یک ویدیوی جدا ضبط کنید
برای توضیح تسکی که انجام داده اید، یا با گوشی خود و یا از طریق رکورد کردن اسکرین لپتاپ، ویدیو ضبط کرده و ضمن توضیح دادن مراحل کار، نتایج آن را نیز نشان دهید؛
🔸 توجه کنید که نتایج تسک حتما باید به صورت واضح نشان داده شوند.
🔸 لطفا در هنگام ارائه، بر روی بخش های بدیهی و غیر ضروری مانور ندهید. اگر توضیحات اضافی داده شود، از نمره ی شما کسر خواهد شد.
🔸 لطفا سعی کنید تایم هر ویدیو کمتر از 5 دقیقه باشد.
🔸 حجم فایل های ویدیویی ممکن است زیاد شود، اما نگران نباشید؛ در ادامه نحوه ی فشرده سازی خودکار توضیح داده خواهد شد.
🔸 اگر کاربر ویندوز یا مک هستید، برای رکورد اسکرین خود می توانید از نرم افزار زیر استفاده کنید (استفاده از هر برنامه ی دیگر ممانعتی ندارد):

3. این ویدیو ها/ویس را از طریقی که ذکر شده کامپرس کرده و برای من ارسال کنید
🔻 پیش نیاز ها:
  • نصب برنامه ی کاربردی FFmpeg
    • در صورتیکه این برنامه را ندارید، می توانید از اینجا دانلود کنید (تقریبا 68 مگ)
    🔸 اگر کاربر Windows هستید، نحوه ی نصب گام به گام در اینجا بیان شده است.
    🔸 اگر کاربر Ubuntu هستید، به روش زیر نصب کنید:
    sudo apt update
    sudo apt install ffmpeg
    🔸 اگر کاربر Mac هستید، به روش زیر نصب کنید:
    brew install ffmpeg --force
    brew link ffmpeg
    برای چک کردن درستی نصب، دستور زیر را در ترمینال وارد کنید:
    ffmpeg -version
  • نصب پکیج پایتون برای FFmpeg
    • pip3 install ffmpy
    یا
    pip install ffmpy
🔻 پس از نصب پیش نیاز ها، فایل زیپ زیر را دانلود کرده و اکسترکت کنید:
Download cvlab-compression
این پوشه حاوی چند فایل و فولدر است:
  • پوشه ی output_
    • پس از اجرای اسکریپت، ویدیو/صوت های کامپرس شده در این فولدر قرار میگیرند و نهایتا کل این فولدر، زیپ خواهد شد؛ (شما با این فولدر کاری نخواهید داشت)
  • پوشه ی tasks
    • در این پوشه، ویدیو ی ضبط شده ی خود از تسک های مربوطه را قرار دهید.
  • پوشه ی questions
    • در این پوشه، ویدیو/صوت مربوط به پاسخگویی سوالات را قرار دهید.
  • فایل compressor.‌py
    • قرار است با اجرای این اسکریپت، فایلهای شما کامپرس شود.
🔻 اسکریپت فشرده سازی را اجرا کنید:
پس از قرار دادن فایلهای مورد نظر در پوشه های مربوطه، ترمینال خود را (در مسیر پوشه ی اصلی) باز کرده و به صورت زیر اسکریپت را اجرا کنید:
python -u compressor.py Alireza-Kavian 9412345
🔸 به جای آرگومان های اول و دوم به ترتیب، اسم کامل و شماره دانشجویی خودتان را قرار دهید.
🔸 توجه کنید که بین نام و نام خانوادگی، space نباشد.
🔸 به هیچ وجه فایل اسکریپت را تغییر ندهید و طبق مطالب گفته شده عمل کنید.
🔸 برای اجرای اسکریپت، ورژن پایتون شما باید 3.4 به بالا باشد.
🔻 فایل های پارت بندی شده را برای من ارسال کنید:
در نهایت، در مسیر پوشه ی اصلی، برای شما یک فایل با پسوند:
.tar.xz
